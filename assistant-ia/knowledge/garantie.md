# Garantie ZotoShop

- Garantie legale de conformite : 2 ans sur la plupart des produits.
- Accessoires (coques) : 1 an.
- La garantie couvre les defauts de fabrication, pas les dommages accidentels ni l'usure.
- Mise en oeuvre : depuis votre espace compte, rubrique Garantie, ou via le support.
- Une extension de garantie a 3 ans est proposee a l'achat sur les ZotoBook.
